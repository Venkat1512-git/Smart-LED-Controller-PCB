# Smart LED Controller PCB

## Overview

This project is a custom-designed Smart LED Controller built around the ATmega328P microcontroller. The project demonstrates the complete embedded hardware development workflow, from schematic capture and PCB layout to firmware development and manufacturing file generation.

## Features

- ATmega328P Microcontroller
- Custom PCB designed in KiCad
- Five LED outputs
- Three Push Button Inputs
- ISP Programming Header
- 16 MHz Crystal Oscillator
- Gerber Files for PCB Manufacturing
- Embedded C Firmware (Coming Soon)
